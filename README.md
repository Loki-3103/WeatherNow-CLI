# WeatherNow CLI Tool

A Python command-line application that fetches and displays **real-time weather data** using the [OpenWeatherMap REST API](https://openweathermap.org/api).

## Purpose
WeatherNow gives users instant access to live weather data directly from the terminal.

## Features
- Real-time weather data via REST API with API key authentication
- Retrieves temperature, humidity, pressure, wind speed, visibility, and weather description
- Structured JSON parsing for clean, formatted output
- Robust error handling for invalid city names, bad API keys, and network failures
